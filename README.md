# Python Digital Clock 🕒

A customizable, real-time Digital Clock application built using Python and Tkinter. This project was developed as part of the Python Application Development Internship at DevStack Technologies.

## 🚀 Features

- **Real-Time Display:** Accurately shows the current system time and date with automatic updates every second.
- **Format Toggling:** Switch seamlessly between 12-Hour and 24-Hour time formats.
- **Customizable Themes:** Includes multiple color themes such as Cyberpunk, Matrix, Dracula, Ocean, Sunset, and more.
- **Dynamic Fonts:** Cycle through various font styles (Helvetica, Impact, Comic Sans, etc.) to personalize the display.
- **Responsive UI:** The clock text automatically resizes to fit the window dynamically.

## 🛠️ Technologies Used

- **Language:** Python 3.x
- **GUI Library:** Tkinter
- **Standard Modules:** `time`



## 💻 How to Run the Project

1.  Ensure you have Python installed on your system.
2.  Clone this repository to your local machine:
    ```bash
    git clone [https://github.com/yourusername/digital-clock.git](https://github.com/yourusername/digital-clock.git)
    ```
3.  Navigate to the project directory:
    ```bash
    cd digital-clock
    ```
4.  Run the application:
    ```bash
    python Digital_clock.py
    ```
